Introduction

  • This is a simple shiny application that shows different types of angles
  • The usage of application is very straightforward: just use slider bar to set the desired angle value and mark the check box to show the angles and their types
  • Angles are measured by degrees

Structure

  • Application consists of two files: server.R and ui.R
  • shiny.R controls the user interface
shinyUI(pageWithSidebar(
  headerPanel("Angles"),
  sidebarPanel(
    p("Use slider bar to set the desired angle value. Mark the check box to show the angles and their types."),
    sliderInput("d", 'Angle size',value = 0, min = 0, max = 360, step = 5,animate=TRUE),
    checkboxInput("l","Show angles",FALSE)
  ),
  mainPanel(
    plotOutput('newPlot')
  )
))
  • server.R contains actual code making the plot [code not included]

Angle types

  • Acute : \((0,90)^{\circ}\)
  • Right angle : \(90^{\circ}\)
  • Obtuse : \((90,180)^{\circ}\)
  • Straight : \(180^{\circ}\)
  • Reflex : \((180,360)^{\circ}\)
  • Complete (perigon) : \(360^{\circ}\)
